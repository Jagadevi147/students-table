# Students Management System (React)

A simple **Students Management System** built using **React.js**.
This project allows users to manage student records with full **CRUD functionality** on the frontend.

## 🚀 Features

- Add new students
- Edit student details
- Delete students with confirmation
- Form validation (Name, Email, Age required)
- Email format validation
- Simulated loading state
- Download student data as Excel
- Data managed using React state (Frontend only)

## 🛠 Technologies Used

- React.js
- JavaScript
- HTML
- CSS
- XLSX (for Excel export)
- FileSaver.js

## 📂 Project Structure

```

1. Navigate to the project folder

```

1. Install dependencies
   npm install

```

1. Start the application
npm start
```
